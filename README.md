# 🛡️ YMAL.space Protection Banner

A lightweight, embeddable security banner for your website. Show visitors your site is protected by YMAL.space.

## 🚀 Quick Embed

Add this single line to your HTML:

```html
<iframe src="https://YOUR_USERNAME.github.io/ymalbanner/" style="position:fixed;top:0;left:0;width:100%;height:40px;border:none;z-index:999999;"></iframe>
